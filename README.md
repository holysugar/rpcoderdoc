# Rpcoderdoc

Generate contract.rb to other document style

## Installation

Add this line to your application's Gemfile:

    gem 'rpcoderdoc'

And then execute:

    $ bundle --binstubs

Or install it yourself as:

    $ gem install rpcoderdoc

## Usage

```
bin/rpcoderdoc some/path/to/contract.rb > rpcoder.textile
```
